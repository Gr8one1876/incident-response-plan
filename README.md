# Capybara Unlimited — incident response plan

**Client:** Capybara Unlimited (simulated organization)
**Document type:** Incident Response Plan (IRP)

---

## Scenario
Capybara Unlimited, a mid-sized company with limited existing security documentation, needed a formal incident response plan reviewed and built out to prepare their team for a security incident. This project involved analyzing their current state and producing an actionable IRP.

## Objective
Review and develop a complete incident response plan covering preparation, detection, containment, eradication, recovery, and post-incident review phases aligned to industry frameworks.

## Tools used
- NIST incident response framework (SP 800-61)
- IR plan documentation and policy writing
- Tabletop exercise scenario development

## Key findings
- Identified gaps in the organization's existing incident response readiness
- Defined clear roles, responsibilities, and escalation procedures
- Established playbooks for the most likely incident types

## Outcome
Delivered a complete incident response plan ready for organizational adoption, covering all six IR phases with defined procedures, communication templates, and escalation paths.

## Skills demonstrated
Incident response planning, NIST framework application, policy writing, playbook development, security documentation, organizational risk assessment
